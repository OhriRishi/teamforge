# FTC TeamForge v2.0.0

**Release Date:** 2025-01-22

## Overview

This major release introduces two new features: **Surveys** for collecting feedback, and **Sheets** for spreadsheet-based data management.

## New Features

### Surveys

Create and manage surveys to collect feedback from team members or the public.

- **Multiple question types**: Short answer, long answer, multiple choice, dropdown, checkboxes, and star ratings
- **Survey templates**: Create reusable branded templates with custom header images and footer content
- **Visibility controls**: Public (anyone), private (authenticated users), or team-only access
- **Response analytics**: View and analyze all survey responses
- **Public sharing**: Share survey links for anonymous response collection

### Sheets

A new spreadsheet feature powered by Univer for managing structured data.

- **Spreadsheet editor**: Full-featured spreadsheet with formulas and formatting
- **Auto-save**: Changes are automatically saved as you work
- **Folder organization**: Organize sheets into folders (separate from notebook folders)
- **Team collaboration**: Sheets are tied to your team and season

## Migration

Run the v2.0.0 migration to enable these features. The migration adds the necessary database tables and columns.

**Estimated time:** 1-2 minutes

**Breaking changes:** None
